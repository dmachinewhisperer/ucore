# Plotting Data from Device

## Basic line plot

```python
import matplotlib.pyplot as plt

data = [23.1, 23.4, 23.2, 23.5, 23.3]  # from device output
plt.figure(figsize=(10, 4))
plt.plot(data)
plt.xlabel("Sample")
plt.ylabel("Temperature (C)")
plt.title("Sensor Reading")
plt.grid(True)
plt.show()
```

## Time-series with timestamps

```python
import matplotlib.pyplot as plt
import numpy as np

data = [23.1, 23.4, 23.2, 23.5, 23.3]
duration_s = 10
t = np.linspace(0, duration_s, len(data))

plt.figure(figsize=(10, 4))
plt.plot(t, data, marker='o', markersize=3)
plt.xlabel("Time (s)")
plt.ylabel("Value")
plt.title("Sensor Over Time")
plt.grid(True)
plt.show()
```

## Multiple sensors

```python
import matplotlib.pyplot as plt

fig, axes = plt.subplots(2, 1, figsize=(10, 6), sharex=True)
axes[0].plot(temp_data, color='red')
axes[0].set_ylabel("Temperature (C)")
axes[1].plot(humidity_data, color='blue')
axes[1].set_ylabel("Humidity (%)")

plt.tight_layout()
plt.show()
```

## Notes
- Always use `plt.show()` to render in Jupyter.
- Use `figsize` for readable plots.
- For real-time-like display, collect all data first, then plot (MicroPython cannot stream to matplotlib).
