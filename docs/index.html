Jupyter notebooks for your microcontroller.

µcore is a Jupyter kernel that runs MicroPython on a microcontroller over USB. Write cells, stream sensor data into live plots, no flashing toolchain required.

1 · Flash your board

Plug in a supported board over USB, then flash µcore directly from this page. Supported today: ESP32, ESP32-S3.

Web Serial requires a Chromium browser (Chrome, Edge, Brave, Opera). Firefox and Safari users can fall back to esptool.py write_flash 0 ucore-<board>.bin.

2 · Install the kernel

  1. In your Jupyter venv: pip install ucore
  2. Launch jupyter lab — the µcore kernel appears in the launcher.
  3. New notebook → start with %%ucore cells; the kernel finds the device automatically.
